B2 · English Grammar

Third
Conditional

If I had studied harder, I would have passed.

if + past perfect would have + p.p.
The idea

Imaginary situations in the past

Used to speculate about things that did not happen — and imagine a different outcome. The past can't be changed.

Examples

If I had studied harder, I would have passed. (I didn't study — I failed)

She would have got the job if she had arrived on time.

The structure

if + past perfect

→ would/could/might have + p.p.

Order is flexible — comma if "if" comes first.

The conditional family

Where does the third fit?

First
if + present → will + base

If it rains, I will stay home. (real future)

Second
if + past → would + base

If I had money, I would travel. (imaginary now)

Third
if + past perfect → would have + p.p.

If I had studied, I would have passed. (imaginary past)

Modal variations

Would, could or might have

If he had saved more money, he would have bought a car. (certain)
If she had studied, she could have passed. (possible)
If we had left earlier, we might have caught the train. (uncertain)
💡 Contractions in speech

'd = had  |  would've = would have

If I'd known, I would've told you.

She'd have come if she'd been invited.

⚠ Watch out

Common mistakes

If I would have known, I would have called.

If I had known, I would have called.

If she arrived on time, she would have got the job.

If she had arrived on time, she would have got the job.

I would have pass if I had studied.

I would have passed if I had studied.

We wouldn't have lost if we would have used the map.

We wouldn't have lost if we had used the map.
