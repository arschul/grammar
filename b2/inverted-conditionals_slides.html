Inverted
Conditionals

Had they used a different street layout, it would have been more difficult.

no if question word order
The idea

Drop the if, swap the order

In formal English we can delete if. The subject and auxiliary then change places, so the clause takes question word order — but it is still a statement.

The transformation

If they had used a different layout… → Had they used a different layout…

If they should decide to go ahead… → Should they decide to go ahead…

Only three

Should + subject + base verb

Were + subject (+ to + inf.)

Had + subject + p.p.

Formal speech and writing only.

1 · Inverted first conditional

Should + subject + base verb

Should they decide to go for a grid pattern, it will produce the following benefits.
Should you require further assistance, please contact our office.
Should the council reject the plan, the developers can appeal.

= if … should — a future possibility the speaker treats as unlikely.

2 · Inverted second conditional

Were + subject (+ to + infinitive)

Were you to use a grid system in a hilly area, you would end up with very steep streets.
Were I in charge of the budget, I would invest far more in public transport.
Were it not for the new bypass, the town centre would still be gridlocked.

= if … were — unreal or improbable present and future.

3 · Inverted third conditional

Had + subject + past participle

Had they used a different street layout, it would have been more difficult to design such a good system.
Had the council listened to residents, the road would never have been built.
Present

Were it not for the flood defences, this district would be under water.

Past

Had it not been for the flood defences, this district would have been under water.

⚠ Watch out

Common mistakes

If had I known, I would have called.

Had I known, I would have called.

Was it not for her advice, I would have failed.

Were it not for her advice, I would have failed.

Should you needed help, call reception.

Should you need help, call reception.

Hadn't he resigned, he would have been promoted.

Had he not resigned, he would have been promoted.
