Inverted Conditionals

What are Inverted Conditionals?

In formal English we can drop if from a conditional sentence. When we do, the subject and the auxiliary verb change places, so the clause takes question word order — but it is still a statement, not a question.

If + subject + auxiliary   →   auxiliary + subject (no if)
If they should decide to go for a grid pattern… → Should they decide to go for a grid pattern…
If they had used a different layout… → Had they used a different layout…

Only three auxiliaries can be inverted this way: should, were and had. Inverted conditionals belong to formal speech and writing — reports, contracts, notices, presentations.

1 · Inverted First Conditional — Should

Used for a future possibility the speaker sees as unlikely or merely hypothetical. It replaces if … should, and often simple if + present too.

Should + subject + base verb,  →  will / can / imperative
Should they decide to go for a grid pattern, it will produce the following benefits.
Should you require further assistance, please contact our office.
Should the council reject the plan, the developers can appeal.

2 · Inverted Second Conditional — Were

Used for unreal or improbable present and future situations. It replaces if … were. With ordinary verbs we need the pattern were + subject + to + infinitive.

Were + subject (+ to + infinitive),  →  would / could / might
Were you to use a grid system in a hilly area, you would end up with very steep streets.
Were I in charge of the budget, I would invest far more in public transport.
Were it not for the new bypass, the town centre would still be gridlocked.

Note the fixed expression Were it not for X (= If it were not for X), used to say that one thing is the only reason another is true.

3 · Inverted Third Conditional — Had

Used for imaginary situations in the past — things that did not happen. It replaces if … had.

Had + subject + past participle,  →  would have / could have / might have + past participle
Had they used a different street layout, it would have been more difficult to design such a good system.
Had the council listened to residents, the road would never have been built.
Had it not been for the flood defences, the whole district would have been under water.

Register and Negatives

Inverted conditionals sound noticeably more formal than if. Compare: If you need help, call me (neutral) and Should you need help, contact reception (formal notice).

In the negative, the contraction is never used. Put not after the subject.

✗ Hadn't he resigned…   ✓ Had he not resigned, he would have been promoted.
✗ Shouldn't you agree…   ✓ Should you not agree with the decision, you may appeal.
⚠ Common mistakes:
✗ If had I known… → ✓ Had I known… (never keep if)
✗ Was it not for her advice… → ✓ Were it not for her advice…
✗ Were the company to expanding… → ✓ Were the company to expand…
✗ Had I knew… → ✓ Had I known… (past participle after had)
✗ Should you needed help… → ✓ Should you need help… (base verb after should)
