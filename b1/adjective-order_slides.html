The key rule

Opinion before fact

Opinion adjectives (beautiful, horrible, lovely) always come first.

Rule 1

Opinion always comes before fact adjectives

1 a beautiful old painting opinion → age
2 a lovely little garden opinion → size
3 an amazing big pink salt lake opinion → size → colour
4 a stunning old French painting opinion → age → origin
Rule 2

Among fact adjectives: Size → Shape → Colour → Origin → Material

1 a long thin silver necklace size → shape → colour
2 a big red Italian sports car size → colour → origin
3 a small square wooden box size → shape → material
4 a huge old brick building size → age → material
Tip

When to use commas

Use a comma between adjectives from the same category. No comma between different categories.

Comma ✓

a friendly, helpful teacher

an amazing, big pink lake

Both are opinions — use a comma.
No comma ✓

a small red bag

a long dark wooden corridor

Different categories — no comma between them.
⚠ Watch out

Common mistakes

a red big car

✓ a big red car

Size comes before colour.

a wooden old French table

✓ an old French wooden table

Age → origin → material.

a pretty, Italian, small vase

✓ a pretty small Italian vase

Opinion → size → origin. No comma between different categories.

a round big silver coin

✓ a big round silver coin

Size comes before shape.

Recap

Remember this

Opinion first
Beautiful, lovely, horrible — always before size, colour, origin, material.
Size → Shape → Colour
a big round red ball — never "a red round big ball".
Origin before Material
an old French wooden clock — age → origin → material.
Commas
Only between adjectives from the same category. Not between different ones.

When adjectives come before a noun, opinion comes first — then facts follow the fixed chain.
